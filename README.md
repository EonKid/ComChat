# ComChat
    ComChat是一款即时聊天工具，主要采用XMPP、HTTP等网络协议，开发一款即时通信软件。
    后台服务器主要基于Openfire进行二次开发，并搭建HFS(HTTP⽂件服务器)⽤于存储⽤户聊天时所发送的图片、语音等⽂件。
    客户端为在iOS系统下基于XMPPFramework框架，开发一款即时通讯软件。

## 主要功能模块:
    * 账号管理模块:注册、登录、明信片等;
    * 好友管理模块:搜索、添加、删除好友;
    * 群组管理模块:创建、搜索、添加、退出群组等;
    * 聊天模块:一对一聊天和群聊,⽀持文本、表情、语音、图⽚等消息类型。
